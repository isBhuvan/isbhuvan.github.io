# Security Policy

Although this is a static portfolio site, security best practices are followed.

---

## 🛡 Supported Versions

Only the `main` branch is actively maintained.

---

## 🚨 Reporting a Vulnerability

If you discover a security issue, please open a GitHub issue with:

- Clear description
- Steps to reproduce (if applicable)
- Suggested mitigation

Sensitive disclosures can be sent privately via email if required.
